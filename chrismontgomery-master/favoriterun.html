FAVORITE RUN

Wireframes, Interface Design, Icon Design, Admin Wireframes

Favorite Run is challenge based run tracking app. Users can create or join community challenge campaigns or simply run for themselves. A challenge campaign gathers users together to run for a common goal; either miles ran, calories burned, or total participants. Certain campaigns will have sponsors, who will donate a dollar amount to a specified charity for certain milestones (eg. $10 for every 100 miles ran).

THE WORK

The trickiest part of design was setting Favorite Run apart from all of the other running apps. Favorite Run is targeted towards people who don't normally run, but are willing to run to support a cause. One thing we wanted to stress with the design was the fun aspect of running. The app uses a bright color scheme and whimsical illustrations and animations to delight users. To get users to return to the app, we added inspirational messages and awarded medals for completing runs.
The other challenge was getting the admin panel to sync to the front facing mobile side. The admin needs the ability to create campaigns, add prizes, setup sponsorships and charities, distribute money, etc. The sheer amount of things the admin can do made creating a clear and easy to use backend a very complex task.
